How to Use This Component

The LLM Selector component can be included in any page to provide model selection functionality.

Include the Component:

<!-- Include the component template -->
<div id="llm-selector-container"></div>
<script src="/static/components/llm-selector.html"></script>

JavaScript Usage:

// Create a selector with default model
const selector = createLLMSelector('gpt-4o');

// Get recommended model for use case
const model = getRecommendedModel('creative');

// Listen for model changes
selector.addEventListener('llm-changed', (e) => {
    console.log('Selected:', e.detail);
});
