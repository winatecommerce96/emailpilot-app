                 Integration Guide
            


            

                
1. Add Required Files

                

                    
<!-- In your HTML <head> -->
<link rel="stylesheet" href="/static/css/strategy-summary.css">

<!-- Before closing </body> -->
<script src="/static/js/strategy-summary-types.js"></script>
<script src="/static/js/strategy-summary-api.js"></script>
<script src="/static/js/strategy-summary-component.js"></script>

                


                
2. Add Container to Your Page

                

                    
<!-- Place this where you want the strategy summary to appear -->
<div id="strategy-summary-container"></div>

                


                
3. Initialize the Component

                

                    
<script>
// Create component instance
const strategySummary = new StrategySummaryComponent('strategy-summary-container');

// Initialize with client ID
await strategySummary.init('your-client-id');

// Or load manually
await strategySummary.load('your-client-id');
</script>

                


                
4. Example Integration in Calendar

                

                    
// Add to your calendar initialization code
async function initCalendar(clientId) {
    // ... existing calendar code ...

    // Load strategy summary
    const strategySummary = new StrategySummaryComponent('strategy-summary-container');
    await strategySummary.init(clientId);
}

                


                

                    Note: The strategy summary will automatically hide if no data is found, so it's safe to initialize on all pages.
