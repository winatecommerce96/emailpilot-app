Custom Variables Test

Test Instructions

  1. Open the Agent Creator Enhanced page
  2. Look for the new "➕ Add Custom Variable" section in the left panel
  3. Click the dropdown arrow to expand the form
  4. Try adding a custom variable (e.g., "my_test_var")
  5. Check that it appears in the variables list
  6. Try using it in a prompt by clicking on it
  7. The variable should no longer show as invalid
  8. Custom variables are saved to browser localStorage

Pre-loaded Custom Variables

The following variables are pre-loaded automatically on first use:

additional_context brand_guidelines target_audience campaign_goals product_focus promotional_offers seasonal_context competitor_insights email_frequency customer_segments historical_performance content_preferences call_to_action exclusion_dates priority_messages

Features

  • Add Custom Variables: Add any variable name you need
  • Categories: Organize variables into categories (Custom, Firestore, System, Integration, Context)
  • Persistence: Variables are saved to browser localStorage
  • Validation: Custom variables are recognized in prompt validation
  • Remove: Click the × next to any custom variable to remove it
  • Auto-complete: Click any variable to insert it into the prompt
